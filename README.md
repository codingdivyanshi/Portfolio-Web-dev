# Modern Web Developer Portfolio

A modern, elegant, and fully responsive personal portfolio website built using HTML5, CSS3, and Vanilla JavaScript.

Designed with a clean user interface, glassmorphism design, smooth animations, and interactive sections to showcase skills, education, projects, and contact information.

---

## Live Demo
Enable GitHub Pages to view the website online.

## Features

- Modern UI Design
- Fully Responsive Layout
- Glassmorphism Components
- Sticky Navigation Bar
- Smooth Scrolling
- Scroll Progress Indicator
- Active Navigation Highlight
- Hero Section
- About Section
- Skills Cards
- Academic Projects
- Education Timeline
- Work Experience
- Contact Form
- Social Media Links
- Mobile Navigation Menu
- Hover Animations
- Ambient Background Effects
- Dynamic Footer Year
